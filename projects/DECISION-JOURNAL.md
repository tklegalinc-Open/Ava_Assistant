# Decision Journal
*Track decisions to counter shiny object syndrome and build pattern recognition*

---

## How to Use This

Before committing to any new project, opportunity, or significant time investment:

1. **Write it down** — forces clarity
2. **Answer the framework questions** — exposes assumptions
3. **Set a review date** — learn from outcomes
4. **Review periodically** — build intuition

---
